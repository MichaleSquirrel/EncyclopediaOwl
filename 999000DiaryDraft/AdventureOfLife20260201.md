## 8.使用tcpdump命令拦截网络请求，并使用wireshark查看来解决问题
### 3.1 tcpdump命令

tcpdump -i any -s 0 host xx.xx.xx.xx and port 8002  -vv -w filename.cap

-i 指定网卡，any表示所有网卡  
host 指定的对端IP（该IP到本机的包）  
post 指定端口  
-vv 显示抓了多少包  
-w抓包文件.cap的保存位置

tcpdump -vv '(host 127.0.0.1 or host 127.0.0.2)' and port 80 -w 1105.cap

tcpdump -vv '(host 127.0.0.1 or host 7.180.200.82)' and port 8900 -w analyzer.cap

忠臣孝子的命 淫乱人的命

姚摩诃：这几天牢A在评论罗翔解读“武松杀嫂案”时提出了一个很有趣的命题：“忠臣孝子的命就是比奸夫淫妇的命值钱”，并且不出意外地，引来了许多法律精英的集体炮轰。然而看过了这些精英们的反驳文章，我发现他们都有意无意地回避了一个关键问题，那就是：如果人和人的命价真的一样，如果法律面前真的人人平等，那为什么现实当中有那么多的同案不同判、同罪不同罚？这些年，法官的自由裁量权已经成了一个冰冷的笑话，以至于最高法三令五申要“切实规范自由裁量权行使”，但效果似乎很一般。法律精英们尽可以高唱“法律面前人人平等”，但事实就是，人和人的命价是不一样的，在法庭上，高贵的命价相对低贱的命价往往具有凌驾性的优势，自由裁量权的圣光总能照耀在他们身上，从而抵扣他的罪过。同样是诬告强奸，在澳门是诬告者坐牢，而内地却是被诬告者判三缓三起步，为何？大陆男人在内地的命价不值钱罢了。其实，就算忠臣孝子的命和奸夫淫妇的命价一个样倒也罢了，只怕在实际判决当中是奸夫淫妇的命比忠臣孝子还高贵，那就绷不住了。
